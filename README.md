# Wireless Water Tank Automation System

## Overview
Developed an automated system to monitor and control water levels in a tank using Arduino and wireless communication.

## Features
- Automatic motor ON/OFF based on water level
- Wireless control using RF communication

## Hardware Used
- Arduino
- Float Switch
- Solid State Relay (SSR)
- 433 MHz RF Transmitter & Receiver
- Water Pump

## Working Principle
- Float switch detects water level
- Arduino processes the signal
- RF module transmits control signal
- Relay controls the motor operation

## Implementation
- Designed control logic for automatic switching
- Integrated RF communication for remote operation
- Ensured safety by preventing overflow and dry run

## What I Learned
- Embedded system design
- Wireless communication (RF modules)
- Real-time control systems
